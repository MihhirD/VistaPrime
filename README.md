## Overview

This project focuses on performing customer segmentation analysis using data analytics and machine learning techniques in Jupyter Notebook. The objective is to identify distinct groups of customers based on their behavior and characteristics, enabling businesses to make informed, data-driven decisions.

The analysis highlights patterns in customer data and provides actionable insights for targeted marketing and improved customer engagement.

---

## Objectives

* Understand customer behavior through data analysis
* Segment customers into meaningful groups
* Identify high-value and low-engagement customers
* Enable targeted marketing strategies
* Support business decision-making with data insights

## Methodology

1. Data Collection

   * Imported dataset containing customer attributes such as demographics and spending patterns

2. Data Preprocessing

   * Handled missing values
   * Normalized and cleaned the dataset
   * Selected relevant features

3. Exploratory Data Analysis (EDA)

   * Visualized trends and distributions
   * Identified correlations between variables

4. Clustering

   * Applied clustering algorithms (e.g., K-Means)
   * Determined optimal number of clusters
   * Grouped customers into segments

5. Analysis

   * Interpreted each segment
   * Derived business insights from clusters
